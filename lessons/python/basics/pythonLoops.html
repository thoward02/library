Loops In Python

Loops

A loop in programming is how you run a task over and over again until it is finished completing its task.

Lets say I wanna get all the apples in a box. Well the logic behind this is "while apples in box: getApple()"

There are 3 different types of loops

While loop

A while loop does a task while two values equal (in most cases) The code for it in python looks like this: 

 x = 1
 y = 1
 while(x==y):
   print('X equals One')
   x = x + 1

So our first two lines defines our variables, then we initiate our while loop by saying while(x==y): . What this says is that while x equals y we will run the next indented code after the :.

After that we pring out some text, then we add one to x, stopping the loop. This stops the loop, because the while loop is no longer true. After this, x now equals 2 and y equals 1, therefor x and y are no longer equal.

Like an if statement we can run code if two values don't equal. Instead of using == we would use != to tell the system that we want these to Not(!) Equal (=).

The code would look like this

 x = 1
 y = 10
 while(x!=y):
   print(x)
   x = x + 1
 print('done!')

This runs the code 9 times, then prints the done message. "But but what if you wanna run it 10 times!", well you would just add one to the loop number. Why? Because when x finally hits the y, the code isn't told to run because x now equals y.

For Loops

What if we want to run code for every item inside of the box? Well the logic would be: "for item in box: runSomeCode()"

In python you often use for loops to run code for every item in a list. Here is an example of for loops: 

 x = [1,2,3,4]
 for numbers in x:
    output = int(numbers) * 2
    print(output)

Now in this code we multiply every item in the list by two. First we define our list of numbers, and set them equal to x. Next is our command that tell the system to run a for loop. We say for numbers in x:, now this command is pretty logical, and the only confusing part is "Numbers". As you run a for loop you go from item to item within the list, running your code on said item. By saying "numbers" we define a variable for the system to put the list section in.

The first time we run the for loop, the "Numbers" variable gets set to equal the first item in the list, in this case it is 1. The second time we run the for loop, the "Numbers" variable gets set to equal 2, because that is what is in the second item in the list. This will go on and go on until there is no more list items in the list. If you have 400 items ina list, you will run your code 800 times

"Numbers" can be changed to what ever you want because it is a variable.

Now there is one thing you haven't seen before, and that is the int() function. This is a python function that turns a variable into a integer. We put the numbers variable inside of it just in case the system thinks it is a string. See in python you can multiply strings. If we were todo 'x' * 100 We would get 100 x's. You see this becomes a problem when you are just looking todo some simple math. You try multipling two variables and you could end up getting what you wanted, the product of the two sums, or you could get an error or a bunch of letters that you did not want.

There are two of these functions. The first is int(), and the second is str(). int() converts variables or characters into a number or integer. str() converts variables or text into a string, which is useful when you want to put a number and word together in a print function. For example ince you can not do print("hey "+1) because python doesn't like letters and numbers together, we can do print("Hey"+str(1)) with the str() converting the number into a string.
